# Home Feature

## Overview

The Home feature handles the landing/main page of the application.

## Structure

- **components/**: React components used in this feature
- **hooks/**: Custom hooks for this feature
- **types/**: TypeScript types specific to this feature
- **services/**: API calls and business logic
- ****tests**/**: Unit and integration tests

## Usage

```typescript
import { HomeHero, useHomeData } from "@/modules/home";
```

## Components

### HomeHero

Main hero section of the home page.

## Hooks

### useHomeData

Fetches data needed for the home page.

## Public API

This feature exports the following:

- Components
- Hooks
- Types
